## 概要

### 問題設定

複数グループの人なるべく効率的にテーブルに配置させるという問題を考えます。

「効率的に」とは以下の条件を満たすことを指すこととします。

１．テーブルの座席数をなるべく埋める（４人掛けのテーブルにはなるべく４人グループを割り当てる）
２．グループが同じ人はなるべく近いテーブルに座らせる
３．他のグループとの距離はなるべく離す
